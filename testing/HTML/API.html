Context API

Context Class

Context serves as the parent to Conductor and Collaborator, and handles the shared behavior, such as initialising styles and creating the top-level window.

Properties

cid: The context identifier.

pages: The list of visible pages, only one is supported for now.

window: The top-level Tk window.

root: The root frame used for laying out the window.

focused_page Index of currently visible page.

Methods

Context(cid): Creates a new context, including a window and root frame.

make_ui_frame(): Creates the user interface widgets for this frame.

Collaborator Class

Collaborator is a specialisation of Context. It is not yet implemented.

Conductor Class

Conductor is a specialisation of Context. It is used by users who browse by themselves or host a shared session.

Methods

Conductor(cid): Create a new Conductor. Loads the homepage automatically.

make_ui_frame(): Creates the user interface for the conductor, which includes navigation buttons and an address entry field.

go_to_page(url): Download a page and display it using the DOM API.

display_collaboration_options(): Displays a window which will let users set sharing options. Not ready yet.

go(url): Add the url to the history and navigate to it.

back()/forward(): Navigate to previously visited pages.

current_page(): Retrieve the currently visible page.
